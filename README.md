Symbolic Regression Benchmarks (Ideal Gas, Rydberg, and Schechter)

•	The file newBenchmarks_dataset.xlsx contains the training datasets for the three benchmark problems: Ideal Gas, Rydberg, and Schechter. These datasets were generated by running the code provided in the original paper, Interpretable Machine Learning for Science with PySR and SymbolicRegression, which is available at: https://github.com/MilesCranmer/pysr_paper.
•	The computation times and the best symbolic regression equations obtained for each random seed are available in the benchmark/taken_results directory.
•	The notebooks for training the Arithmetic Neural Network are provided in the corresponding benchmark folders:
o	IdealGas_model.ipynb
o	Rydberg_model.ipynb
o	Schechter_model.ipynb
Each notebook trains the Arithmetic Neural Network on its respective benchmark dataset.


KAN-ODEs Benchmark
•	The original implementation and source code are available at: https://github.com/DENG-MIT/KAN-ODEs
•	Original paper: KAN-ODEs: Kolmogorov–Arnold Network Ordinary Differential Equations for Learning Dynamical Systems and Hidden Physics.
•	To train the Arithmetic Neural Network, use the notebook predator_prey_model.ipynb in Google Colab.
•	To compare the Arithmetic Neural Network with the original KAN-ODEs implementation and reproduce the comparison figures, use comparison_original&ArithmeticPapers.ipynb, located in the /Lotka-Volterra-Pytorch/efficient_kandirectory (alongside efficientkan.py).
